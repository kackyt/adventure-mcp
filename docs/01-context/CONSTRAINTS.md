# CONSTRAINTS.md - 制約事項

## 1. チーム体制とリソース
- 開発、プロダクトリード(自分): 1名 (スキルレベル: 上級, 稼働率: 100%)

## 2. 予算制約
- 開発予算: なし
- 運用予算（月額）: なし
- 外部サービス利用可能額: 月額10$(Google Cloud等)

## 3. 技術的制約
- バックエンド: Node.js (TypeScript) - MCP実装のしやすさ、InkJSとの連携のため
- シナリオエンジン: Ink (inkle) - シナリオの分岐管理、デッドエンドの静的解析のため
- フロントエンド: なし - AIエディタに絞るため(デバッグ用cliは用意する)
- データベース: なし - いったんファイル保存で十分
- インフラ: なし - Webサービスとしては提供しない

### 技術的に不安な点
- Ink言語自体の学習コスト（独自の複雑な実装を保守するよりは安全）
- MCPサーバー（TypeScript）とInkJS（状態管理）のシームレスな繋ぎ込み
- 生成AIのプロンプトエンジニアリング（AIを優秀なゲームマスターにするための調整）
