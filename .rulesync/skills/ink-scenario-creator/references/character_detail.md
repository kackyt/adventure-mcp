# 登場人物の設定例

## 主人公(プレーヤー) 1 (引退した元刑事)
- 職業: 引退した元刑事
- 被害者との関係：古い友人
- できること：
  - 前職のコネを生かして警察から情報を得る
  - 聞き取り
  - 証拠品の回収・警察への提出・容疑者につきつける
- できないこと：
  - 事件現場に許可なく立ち入ること
  - 逮捕、鑑識など警察がかかわること
- 捜査する経緯：
  古くからの知人が自死したとの連絡を受け、急遽故郷へ帰って現場へ駆けつける。前職のコネを使って事件の状況を聞くが、不可解な点を見つけて独自に調べることにした。

## 主人公(プレーヤー) 2 (私立探偵)

- 職業: 私立探偵
- 被害者との関係：たまたま事件にでくわした
- できること：
  - 聞き取り
  - 証拠品の回収・警察への提出・容疑者につきつける
- できないこと：
  - 事件現場に許可なく立ち入ること
  - 逮捕、鑑識など警察がかかわること
  - 警察から情報を得る行為
- 捜査する経緯：
  温泉宿に休暇をとっていたところ、事件に巻き込まれる。警察は自死とみて捜査を打ち切ろうとしていたが、主人公には気になる点がいくつかみられたため、独自にしらべていく。
